Intro to CASE Statement

The case statement is used for conditional expressions in SQL. It behaves just like if else statements and conditional flow in programming.

General Syntax

In general, here is the syntax for a general case statement in SQL

CASE
    WHEN Condition1 THEN result1
    WHEN Condition2 THEN result2
    WHEN ConditionN THEN resultN
    ELSE result END as column_description

The case statement goes through the cases and returns a result as soon as a condition is met. As soon as a condition is met it breaks out and the result is returned. That is, it will not attempt anymore conditions once one is met. If no conditions are met then it will use the result from the ELSE clause. If there is no ELSE clause, then a NULL record will be returned.

Example 1

SELECT
  duration_minutes,
  CASE
    WHEN duration_minutes <= 5 THEN 'less_than_5_minutes'
    WHEN duration_minutes > 5
  AND duration_minutes <= 10 THEN '5_to_10_minutes'
    WHEN duration_minutes > 10 AND duration_minutes <=30 THEN '10_to_30_minutes'
    ELSE 'greater_than_30_minutes'
  END AS trip_duration
FROM
  `bigquery-public-data.austin_bikeshare.bikeshare_trips`
LIMIT
  1000

Example 2

  • you could try an aggregation like this:
SELECT
  CASE
    WHEN duration_minutes <= 5 THEN 'less_than_5_minutes'
    WHEN duration_minutes > 5
  AND duration_minutes <= 10 THEN '5_to_10_minutes'
    WHEN duration_minutes > 10 AND duration_minutes <=30 THEN '10_to_30_minutes'
    ELSE 'greater_than_30_minutes'
  END AS trip_duration,
  COUNT(*) AS cnt
FROM
  `bigquery-public-data.austin_bikeshare.bikeshare_trips`
GROUP BY
  trip_duration
